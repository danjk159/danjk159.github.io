sudo apt-get install postgresql
安装完之后会创建一个账户postgres,修改密码 sudo passwd postgres
修改成功后，切换到postgres
su postgres
进入之后新增一个用户和数据库用来在原用户登录,在此用我自己的用户名
createuser zxw
createdb zxw
返回自己的命令行
exit
在命令行中进入postgreSQL命令行环境
psql
在此，安装postgreSQL结束

安装phppgadmin
sudo apt-get install phppgadmin
将phppgadmin的文件放置在apache服务器目录下,我这是软链接连的
cd var/www sudo ln -s /usr/share/phppgadmin
修改密码验证方式
sudo gedit /etc/postgresql/9.1/main/pg_hba.conf
搜索 "# IPv4 local connections:host" ,将其中的"md5"数值改为"trust",否则会web登录时会密码验证不通过
安装pdo_pgsql，得安装pdo_pgsql才能通过phppgadmin访问postgreSQL数据库
sudo apt-get install postgresql-server-dev-8.3
pecl search pdo(这一步不是必须的)
sudo pecl install pdo_pgsql
在浏览器中输入http://localhost/phppgadmin/然后输入帐号密码即可访问
